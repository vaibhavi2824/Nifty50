📘 About the Nifty Return Predictor

This project uses historical stock data from Nifty 50 to predict short-term returns based on lag features and linear regression.

  • ✅ Built with: Python, Flask, Scikit-learn
  • 📊 Charting with: Chart.js
  • 🌗 Dark/Light mode toggle with localStorage
  • 📁 CSV Upload and real-time model retraining
  • 🧪 Shows R² Score & MSE for model evaluation
  • 📜 Swagger API Documentation enabled

Created by Vaibhavi Gurav during internship. You can use, extend, or deploy it for educational or real-world stock analysis purposes.
